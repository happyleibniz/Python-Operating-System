## Overview

## 项目结构
> - `font/`:字体文件夹 :capital_abcd:
> - `images/`:图片文件夹 :flower_playing_cards:
> - `software/`:系统应用文件夹 :computer:
>   - `code/`:用于存储exe应用的源代码文件夹 :computer:
> - `system/`:用于存放系统设置变量或其他配置文件的文件夹 :computer:
> - `Disk/`:虚拟磁盘文件夹 :floppy_disk:
